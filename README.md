# ys-git-version-alpha

Git の使い方の練習のために作成したリポジトリになります。(※ α 版)  
こちらのリポジトリは、fork さえすれば壊しても問題ありません。  
もし Git 操作に失敗した場合は、再度このリポジトリを Fork してもらえれば 0 からやり直せます。

## このカリキュラムで学べること

- Git に関する基礎的な知識
- 実務での Git の扱う流れ
- Git を使用するときに使うコマンド
